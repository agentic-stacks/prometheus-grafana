# prometheus-grafana

An [agentic stack](https://github.com/agentic-stacks/agentic-stacks) that teaches AI agents how to operate prometheus-grafana.

## Usage

```bash
agentic-stacks init agentic-stacks/prometheus-grafana my-project
cd my-project
agentic-stacks pull
```

Then start Claude Code — it reads `.stacks/prometheus-grafana/CLAUDE.md` and becomes an expert operator.
